# Instrucciones para configurar el proyecto demo

Sigue estos pasos para configurar y ejecutar el proyecto:

1. Descarga e instala CUDA desde [este enlace](https://pytorch.org/).
   
2. Instala las dependencias del proyecto ejecutando el siguiente comando en tu terminal:

    ```bash
    pip install -r requirements.txt
    ```

3. Una vez que hayas instalado las dependencias, puedes ejecutar el script principal utilizando el siguiente comando:

    ```bash
    python main.py
    ```

# Instrucciones para configurar el proyecto frontend_react

1. Instala node > v.16 [este enlace](https://nodejs.org/en/download/current).

2. abre la carpeta frontend_react

    ```bash
    cd ./frontend_react
    ```

2. npm i

3. npm run start
  

¡Eso es todo! Ahora estás listo para ejecutar el proyecto en tu entorno de desarrollo.
